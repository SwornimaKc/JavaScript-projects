 Web Development Mini Projects
This repository contains simple HTML projects created for learning and practice.
Projects Included

1. Calculator

   * A basic calculator built using HTML, CSS, and JavaScript.
   * Performs simple arithmetic operations.

2. To-Do List

   * A simple task management web app.
   * Users can add and remove tasks.

3. Rock Paper Scissors Game

   * A browser-based game where the user plays against the computer.

Technologies Used

* HTML
* CSS
* JavaScript

Purpose

These projects were created to practice basic web development concepts such as UI design and event handling.
